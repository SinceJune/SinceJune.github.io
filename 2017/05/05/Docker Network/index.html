Docker Network

2017.05.05 John Doe  热度 ℃

在Linux下，Docker使用内核的Network Namespace实现资源隔离，不同的Network Namespace有各自的网络设备、协议栈、路由表、防火墙规则等。通过对Network Namespace的操作，Docker实现了五种容器网络模式。

  1. none

    不配置网络， 需要设置加--net=none启动容器，启动后，容器只有lo环回接口，但是仍然可以手动配置网络。
